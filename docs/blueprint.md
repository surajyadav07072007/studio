# **App Name**: VeriJob

## Core Features:

- Job Details Input: Allow students to enter job details such as company name, job/internship links, recruiter email, and job description.
- Website Security Check: Check if the website is secure (HTTPS), its age, and any suspicious elements.
- Email Verification: Verify the recruiter's email, checking for free email services (Gmail, etc.) and consistency with the company name.
- Job Description Analysis: Analyze the job description for suspicious elements such as unrealistic salary, urgent joining, or requests for fees.
- Risk Prediction: Use machine learning tool to predict the risk level (safe, be careful, fake) of a job based on previous fake job data and the current job data.
- Trust Score Generation: Generate a trust score (0-100) based on the analysis and checks performed, indicating the job's safety level.
- Student Reporting System: Allow students to report fake jobs, upload proof, and contribute to the database of fake job patterns.
- Dashboard: Display of analyzed job/internship postings with history of past analyses.

## Style Guidelines:

- Primary color: Deep sky blue (#4169E1), representing trust and security.
- Background color: Very light blue (#E6EFFF), to create a calm and trustworthy environment.
- Accent color: Sea green (#2E8B57), for CTAs and highlights.
- Body and headline font: 'Inter', sans-serif, for a modern, objective feel.
- Use simple and clear icons to represent each step of the verification process.
- Clear and user-friendly layout with distinct sections for input, analysis, and results.
- Subtle animations and transitions to guide the user through the verification process.
